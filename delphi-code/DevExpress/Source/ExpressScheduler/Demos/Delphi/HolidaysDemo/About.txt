In this demo you can:

 - Create/delete/edit events (use popupmenus, inplace editor and dialogs)

 - See and use different integrated popupmenus for different scheduler areas

 - Choose different views (View menu: Day/Work week/Week/Month/Timegrid/Year. As an alternative for the Day/Work week/Week/Month select days in the Date Navigator)

 - Change the scheduler's layout (View menu: Date Navigator/Control Box/View position; you can also use the splitters located between the View, Date Navigator and Control Box)

 - See how the scheduler changes its appearance depending on the Look&Feel used (Look&Feel menu)

- Menu File/Export:
	- export data in either Excel's format, as text, html or xml

- Menu File/Synchronization:
	- synchronize the scheduler with MS Outlook®
	- synchronize MS Outlook® with the scheduler

- Menu Options/Day:
	- hide the all-day event area
	- make event times permanently visibile regardless of whether they match the timescale's units or not
	- hide non-working hours
	- show minutes in a time ruler
- Menu Options/Week:
	- compress weekends
	- switch between either one or two columns of week view days
- Menu Options/Month:
	- compress weekends
	- hide weekends
- Menu Options/Timegrid:
	- hide non-working hours
- Menu Options/Year:	
	- show only all-day events

- Menu Events/Intersection:
	- schedule overlapped events
- Menu Events/Sharing between resources:
	- allow events to be assigned to specific resources

- Menu Holidays/Highlight color
	- highlight holiday dates in the Date Navigator with the selected color
- Menu Holidays/Show hints
	- show or hide hints for holiday dates displayed in the Date Navigator

- "Holidays editor" button
	- initialize the Holidays editor. With the Holidays editor, an end-user can add new holidays, import or export holidays from/to external sources

- "Generate Holiday Events" button
	- schedule holiday events for selected resources
