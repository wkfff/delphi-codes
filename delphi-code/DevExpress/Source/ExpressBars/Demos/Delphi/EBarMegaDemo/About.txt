Each of these samples (Office, IE, Database) has its own set of toolbars to help illustrate different aspects of functionality available within ExpressBars.

See Menu\Options for display facilities where you can  experiment with different Side Bar and Bars Options. In particular, try changing between the five styles: 
Standard, Enhanced, Flat, XP and Office11.
